HTIR or short for Human Trafficking is Real a campaign Project for The school exhibition. The project is an informative website that explains human trafficking in South Africa and presents the Stats in a unique way using Augmented Reality (AR) technology.

The way it works is that in the desktop website there is a map that wants the user to scan using the app and the South African Map will pop up selecting a province will show the stats.
